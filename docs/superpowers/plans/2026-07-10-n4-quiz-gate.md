# N4 Quiz Gate Mechanic Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give the staircase (the `final-quiz` interactive, already gating on lesson completion) a real 3-attempt/24h-cooldown mechanic with placeholder quiz content, a permanent passed state, and a celebratory sparkle burst on pass.

**Architecture:** New localStorage-backed state (`nekoBunko.n5.quizGate`: attempts used + lockout timestamp) drives a new branch inside the existing `openPanel()` function. "Passed" reuses the existing `progress['final-quiz']` completion flag — no new flag, no changes to the staircase's existing locked/available/completed visuals. A new hidden-by-default "Fail" button is added to the shared panel DOM. A new `spawnPassSparkle()` scene method fires once on Pass, using the same lightweight emoji+tween pattern already used for the glow/checkmark icons elsewhere in this file.

**Tech Stack:** Phaser 3.90.0 (already in use), vanilla JS, `localStorage` (matching the existing `saveProgress()`/`saveCatColor()` degrade-to-session-only pattern), no new dependencies.

**Design spec:** [docs/superpowers/specs/2026-07-10-n4-quiz-gate-design.md](../specs/2026-07-10-n4-quiz-gate-design.md)

## Global Constraints

- Mechanic only — no real quiz content. "Pass (test)"/"Fail (test)" buttons stand in for a real quiz.
- No N4 navigation — no N4 page exists yet. Passing shows a permanent "You passed! N4 is coming soon." message, never re-locks.
- No new "passed" state — reuses `progress['final-quiz']` exactly as every other completion flag works today.
- No live-ticking countdown — the lockout message is computed fresh at each click, not updated in real time.
- Cooldown expiry is lazy (checked on next interaction), not a background timer.
- New localStorage key `nekoBunko.n5.quizGate`, independent of `nekoBunko.n5.progress` and `nekoBunko.n5.catColor`. Same try/catch degrade-to-session-only pattern as the rest of this codebase.
- No changes to shelves or review-1/review-2 — the new logic is scoped to `entry.id === 'final-quiz'` only.
- All work lives in the existing single-file pattern (`assets/js/n5-phaser-game.js`), matching every prior sub-project this session.

```js
function loadQuizGateState() {
  try {
    const raw = localStorage.getItem(QUIZ_GATE_KEY);
    if (!raw) return { attemptsUsed: 0, lockedUntil: null };
    const parsed = JSON.parse(raw);
    return {
      attemptsUsed: typeof parsed.attemptsUsed === 'number' ? parsed.attemptsUsed : 0,
      lockedUntil: typeof parsed.lockedUntil === 'number' ? parsed.lockedUntil : null,
    };
  } catch (e) {
    return { attemptsUsed: 0, lockedUntil: null };
  }
}

function saveQuizGateState(state) {
  try {
    localStorage.setItem(QUIZ_GATE_KEY, JSON.stringify(state));
  } catch (e) {
    // localStorage unavailable — degrade to session-only, same pattern
    // as saveProgress()/saveCatColor().
  }
}

function formatLockMessage(msRemaining) {
  const totalMinutes = Math.max(1, Math.ceil(msRemaining / 60000));
  const hours = Math.floor(totalMinutes / 60);
  const minutes = totalMinutes % 60;
  return `Locked - try again in ${hours}h ${minutes}m`;
}

// Applies lazy cooldown-expiry reset (spec: "attempts reset to 3... no
// re-review required" once 24h has passed), then returns what the UI needs.
function getQuizGateStatus() {
  const state = loadQuizGateState();
  if (state.lockedUntil !== null && Date.now() >= state.lockedUntil) {
    state.attemptsUsed = 0;
    state.lockedUntil = null;
    saveQuizGateState(state);
  }
  const locked = state.lockedUntil !== null && Date.now() < state.lockedUntil;
  return {
    state,
    locked,
    attemptsLeft: Math.max(0, QUIZ_MAX_ATTEMPTS - state.attemptsUsed),
    lockMessage: locked ? formatLockMessage(state.lockedUntil - Date.now()) : null,
  };
}
```

```js
  openPanel(entry) {
    const state = entry.kind === 'shelf'
      ? getState(entry.id, entry.prereq, this.progress)
      : (this.progress[entry.id] ? 'completed'
        : (entry.requires.every((r) => this.progress[r]) ? 'available' : 'locked'));

    if (state === 'locked') {
      showToast('Not yet…');
      return;
    }

    if (entry.id === 'final-quiz' && state !== 'completed') {
      const gate = getQuizGateStatus();
      if (gate.locked) {
        showToast(gate.lockMessage);
        return;
      }
      this.openQuizGatePanel(entry, gate);
      return;
    }

    const panel = ensurePanel();
    document.getElementById('nekoPanelTitle').textContent = entry.title;
    document.getElementById('nekoPanelBody').textContent = state === 'completed'
      ? (entry.id === 'final-quiz' ? 'You passed! N4 is coming soon.' : 'Already completed. Review anytime.')
      : (entry.kind === 'shelf' ? 'Lesson content coming soon.' : 'Checkpoint content coming soon.');
    panel.style.display = 'flex';

    const completeBtn = document.getElementById('nekoPanelComplete');
    const closeBtn = document.getElementById('nekoPanelClose');
    const failBtn = document.getElementById('nekoPanelFail');
    failBtn.style.display = 'none';
    completeBtn.style.display = '';
    completeBtn.textContent = 'Complete';
    const onComplete = () => {
      this.progress[entry.id] = true;
      saveProgress(this.progress);
      this.refreshAllStates();
      this.closePanel();
    };
    const onClose = () => this.closePanel();
    completeBtn.onclick = onComplete;
    closeBtn.onclick = onClose;

    this.panelOpen = true;
  }

  // Only reached for the not-yet-passed final-quiz case, after openPanel()
  // has already confirmed lessons are complete and the gate isn't locked.
  openQuizGatePanel(entry, gate) {
    const panel = ensurePanel();
    document.getElementById('nekoPanelTitle').textContent = entry.title;
    const n = gate.attemptsLeft;
    document.getElementById('nekoPanelBody').textContent =
      `You have ${n} attempt${n === 1 ? '' : 's'} remaining.`;
    panel.style.display = 'flex';

    const completeBtn = document.getElementById('nekoPanelComplete');
    const closeBtn = document.getElementById('nekoPanelClose');
    const failBtn = document.getElementById('nekoPanelFail');

    completeBtn.style.display = '';
    completeBtn.textContent = 'Pass (test)';
    failBtn.style.display = '';

    completeBtn.onclick = () => {
      this.progress[entry.id] = true;
      saveProgress(this.progress);
      this.refreshAllStates();
      this.closePanel();
      this.spawnPassSparkle(entry.x, entry.y);
    };

    failBtn.onclick = () => {
      const state = loadQuizGateState();
      state.attemptsUsed += 1;
      if (state.attemptsUsed >= QUIZ_MAX_ATTEMPTS) {
        state.lockedUntil = Date.now() + QUIZ_LOCKOUT_MS;
        saveQuizGateState(state);
        this.closePanel();
        showToast('Locked for 24 hours.');
      } else {
        saveQuizGateState(state);
        this.closePanel();
        showToast(`Try again (${QUIZ_MAX_ATTEMPTS - state.attemptsUsed} left)`);
      }
    };

    closeBtn.onclick = () => this.closePanel();

    this.panelOpen = true;
  }
```
