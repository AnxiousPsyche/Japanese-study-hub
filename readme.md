# JP Library OS

A retro "desktop OS" web app for learning Japanese (JLPT N5/N4), built as a static, no-build multi-page site (plain HTML/CSS/vanilla JS).

Start here: **[CLAUDE.md](./CLAUDE.md)** — architecture overview, how to run it locally, and the current state of the codebase. That file is kept up to date as the primary reference; treat other docs in this repo (`SUMMARY.md`, `JP_Library_OS_Project_Summary.md`, `Neko-Bunko-Cat-Library-Spec.md`, `docs/superpowers/`) as historical design notes rather than current specs.
